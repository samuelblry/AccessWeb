# PROJET RESIDUE_ - Mini-site E-commerce Accessible

## 📝 Description du projet
Ce projet est un mini-site e-commerce développé en **HTML, CSS et JavaScript Vanilla** (sans framework) dans le cadre du module d'Accessibilité Web.

Le thème choisi est **RESIDUE_**, une marque de vêtements streetwear au style brutaliste. L'objectif principal était de concilier une esthétique forte (design brut) avec les normes d'accessibilité WCAG.

## 🚀 Fonctionnalités Clés
* **Navigation Accessible :** Support complet de la navigation au clavier (Tabulation).
* **Page Produit :** Carrousel d'images contrôlable et vidéo de présentation avec sous-titres (.vtt).
* **Formulaire de Contact :** Validation stricte en JavaScript, aide à la saisie et gestion des erreurs.
* **Modale Accessible :** Gestion du "Focus Trap" (piège à focus) et fermeture via la touche Échap (Escape).
* **Design Responsive :** Adaptation mobile et desktop.

## 🛠️ Stack Technique
* HTML5 (Sémantique et WAI-ARIA)
* CSS3 (Mise en page Flexbox/Grid)
* JavaScript (Vanilla ES6)

## ✅ Validation Qualité
* **W3C Validator :** Code HTML valide.
* **Lighthouse :** Score > 90 en Accessibilité et Bonnes Pratiques.
* **WAVE :** Aucune erreur de contraste critique.
